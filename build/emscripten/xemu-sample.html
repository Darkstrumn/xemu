Xemu in-browser demonstration page

Xemu emulator demonstrations compiled from Xemu source with the emscripten emulator, so you can try them with only a web broser. WORK-IN-PROGRESS
NOTE: this requires relative decent browser, almost no hope for IE, and probably even Safari is not decent enough (?), I would recommend up-to-date version of Firefox or Chrome, probably (what we need: HTML5 canvas support, asm.js and WebAssembly support).
